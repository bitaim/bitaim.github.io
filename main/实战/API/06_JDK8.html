# Base64

之前Base64加解密用的是java中 sun.misc.BASE64Encoder。Base64是一种基于64个可打印字符（A-Z,a-z，0-9，+，/）来表示二进制数据的方法，能够表示任意二进制数据。通常应用在图片的编解码，密码加解密。网站中登录，密码不会明文传输，通过编码传递到后台，这样别人抓包也看不到密码的原文。

public class Base64Test {
    public static void main(String[] args) {
        Base64.Encoder encoder = Base64.getEncoder();
        Base64.Decoder decoder = Base64.getDecoder();
        // 原密码
        String blankPassword = "123456";
        // 加密后
        String secretPassword = encoder.encodeToString(blankPassword.getBytes("UTF-8"));
        // 解密后
        String password = new String(decoder.decode(secretPassword),"UTF-8");
    }
}

# 日期类

JDK8之前使用的日期常用类，SimpleDateFormat（线程不安全）、Date、Calendar，API设计比较差，日期/时间对象比较，加减麻烦。

JDK8之后，新增日期类，LocalDateTime、LocalDate、LocalTime、Duration。

public class JDKTimeTest {
    public static void main(String[] args) {
        // 目前日期 2019-12-07
        LocalDate today = LocalDate.now();
        // 获取年份
        today.getYear();
        // 获取月份（英文）
        today.getMonth();
        // 获取月份（数字）
        today.getMonthValue();
        // 获取几号
        today.getDayOfMonth();
        // 获取周几
        today.getDayOfWeek();
        // 年份加1，反对的时间对象才是+1的，today不变。
        LocalDate changeDate = today.plusYears(1);
        // 日期比较
        changeDate.isAfter(today);
        // 日期格式化
        DateTimeFormatter dtf = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
        LocalDateTime time = LocalDateTime.now();
        String timeStr = dtf.format(time);
        // 获取指定日期的时间对象
        LocalDateTime ldt1 = LocalDateTime.of(2020,11,11,8,20,30);
        // 计算日期时间差
        LocalDateTime ldt2 = LocalDateTime.now();
        Duration duration = Duration.between(ldt1,ldt2);
        // 相差天数
        duration.toDays();
        // 相差小时
        duration.toHours();
        // 相差分钟
        duration.toMinutes();
    }
}

# Optional

主要解决空指针问题。

public class OptionalTest {
    public static void main(String[] args) {
        User user = null;
        // 如果of中传入的是null，则会报空指针
        Optional.of(user);
        // 如果不确定对象是不是null
        Optional<User> optional = Optional.ofNullable(user);
        if (optional.isPresent()) {
            user = optional.get();
        }
        // 如果对象为null，返回默认对象
        Optional.ofNullable(user).orElse(new User());
        int age = Optional.ofNullable(user).map(user->user.getAge()).orElse(7);
    }
}
