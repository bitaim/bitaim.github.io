    # SPI

      SPI技术被广泛应用在各种开源框架中，比如dubbo、Sharding等，为应用开发者改造框架提供了更加的方式。

    # 定义接口

    package com.bitaim.study.spi;

public interface HelloService {

    void sayHello();

}

    # 实现类

    package com.bitaim.study.spi;

public class ChineseHelloServiceImpl implements HelloService {

    @Override
    public void sayHello() {
        System.out.println("你好");
    }

}

    package com.bitaim.study.spi;

public class EnglishHelloServiceImpl implements HelloService {

    @Override
    public void sayHello() {
        System.out.println("hello");
    }

}

    # 配置SPI

      在resources目录下创建META-INF/services文件夹，创建 com.bitaim.study.spi.HelloService为名称的文件，内容填写两个实现类的全限命名即可。

    com.bitaim.study.spring.spi.ChineseHelloServiceImpl
com.bitaim.study.spring.spi.EnglishHelloServiceImpl

    # 加载

    import com.bitaim.study.spi.HelloService;
import org.junit.Test;

import java.util.ServiceLoader;

public class SpiTest {

    @Test
    public void test01() {

        ServiceLoader<HelloService> helloServices = ServiceLoader.load(HelloService.class);
        helloServices.forEach(HelloService::sayHello);

    }

}
