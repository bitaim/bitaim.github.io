    # 删除数据的相关问题

    1、删除数据磁盘空间并不会缩小？

      当删除数据时，只会把数据标记为删除。当有数据插入到这个数据页时，就会复用这个位置，所以磁盘文件的大小不会缩小。

      当删除整个数据页数据的时候，那么整个数据页就可以被复用了。即使删除整个表的数据，磁盘文件大小也不会缩小，只是所有的数据页都可复用了。如果相邻两个数据页利用率都很小，就会把这两个数据页的数据合并到一个数据页，把另外一个数据页标记为可复用。

    2、避免同时删除过多数据

      会造成cpu利用率过高，从而影响别人对数据库的访问。可以分批次删除。
